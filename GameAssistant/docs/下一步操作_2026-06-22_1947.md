## P13 — LabelImg 标注（约 30 分钟）

### 安装 LabelImg

```bash
pip install labelImg
```

### 设置预定义类别

编辑 `datasets/raw/predefined_classes.txt`，内容改为：

```
quest_marker
```

### 标注流程

```bash
# 切换到项目根目录
cd c:\project\model-training\GameAssistant

# 启动 LabelImg
labelImg datasets\raw_yolo\quest_marker
```

标注操作：
1. 打开图片目录 `datasets/raw_yolo/quest_marker/`
2. 加载 `datasets/raw/predefined_classes.txt`（File → Predefined Classes）
3. 对每张图，用矩形框框住 NPC 头顶的 `?` 或 `!` 标记
4. 标注类别选择 `quest_marker`
5. 保存（自动生成同名 `.txt`，与图片同目录）
6. 下一张（快捷键 `D`）

### 标注质量检查

完成后抽查几张 `.txt` 文件，确认格式正确：

```
# YOLO 格式：class_id x_center y_center width height（归一化 0~1）
0 0.512 0.234 0.045 0.078
```

## P15 — ElementDetector 联调测试

### 测试模板匹配

```bash
python -c "
from pathlib import Path
import cv2
from pc.core.screenshot_collector import ScreenshotCollector
from pc.core.device_profile import DeviceProfile
from pc.core.template_matcher import TemplateMatcher

sc = ScreenshotCollector()
frame = sc.capture_frame()
if frame is None:
    print('截图失败，检查 adb 连接')
    exit(1)

d = DeviceProfile()
m = TemplateMatcher(Path('templates'), d)
results = m.match_all(frame)

print(f'截图尺寸: {frame.shape[1]}x{frame.shape[0]}')
print(f'设备: {d}')
print(f'检测结果: {len(results)} 个')
for r in results:
    print(f'  [{r.source}] {r.cls} conf={r.conf:.3f} at ({r.x},{r.y})')
"
```

### 测试 YOLO 推理

```bash
python -c "
from pathlib import Path
import cv2
from pc.core.screenshot_collector import ScreenshotCollector
from pc.core.element_detector import ElementDetector

sc = ScreenshotCollector()
frame = sc.capture_frame()
if frame is None:
    print('截图失败')
    exit(1)

detector = ElementDetector(
    template_dir=Path('templates'),
    yolo_model_path=Path('ml/weights/yolo/best.pt'),
)
results = detector.detect(frame)
print(f'检测结果: {len(results)} 个')
for r in results:
    print(f'  [{r.source}] {r.cls} conf={r.conf:.3f}')
"
```

### 常见问题排查

| 症状 | 原因 | 解决 |
|---|---|---|
| `FileNotFoundError: 设备配置文件` | `config/devices.json` 不存在 | 确认在项目根目录运行 |
| 模板全部匹配失败 | 截图分辨率与参考分辨率差异大 | 检查 `DeviceProfile` 输出的 scale 值 |
| YOLO 全空 | 模型未训练/路径错误 | 确认 `ml/weights/yolo/best.pt` 存在 |
| `capture_frame` 返回 None | adb 连接断开 | `adb kill-server && adb start-server` |
| 模板匹配只有 1 个结果 | NMS 去重过度 | 调低 `template_threshold`（game.yaml）|
