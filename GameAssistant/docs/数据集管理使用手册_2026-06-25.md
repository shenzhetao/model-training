> 文档版本：v1.0
> 生成日期：2026-06-25
> 适用版本：GameAssistant Web

## 一、模块概述

数据集管理模块用于组织和管理 YOLO 模型训练所需的数据集。一个完整的数据集包含多张图片及其对应的标注信息，模块支持版本管理、数据增强、数据划分等功能。

模块包含以下主要功能区域：

| 区域 | 功能说明 |
|------|----------|
| 数据集列表 | 左侧展示所有已创建的数据集 |
| 数据集详情 | 显示数据集的基本信息 |
| 版本管理 | 管理数据集的多个版本 |
| 版本对比 | 对比不同版本的数据集差异 |

---

## 二、核心概念

### 2.1 数据集（Dataset）

数据集是一个容器，用于组织特定主题或项目的所有训练数据。例如：
- `game_ui_v1`：游戏界面识别数据集
- `mobile_game_buttons`：手机游戏按钮数据集

### 2.2 版本（Version）

数据集支持多版本管理，每个版本可以有不同的：
- 图片集合
- 数据划分比例
- 数据增强配置
- 类别配置

### 2.3 数据划分

YOLO 训练需要将数据划分为：
- **训练集（train）**：模型学习的主要数据
- **验证集（val）**：调整超参数的数据
- **测试集（test）**：最终评估的数据

## 四、各功能详细说明

### 4.1 新建数据集

**用途**：创建一个新的数据集容器。

#### 操作步骤

1. 点击页面右上角的「**新建数据集**」按钮
2. 填写数据集信息：
   - **数据集名称**：英文名称，如 `game_ui_v1`
   - **描述**（可选）：数据集的详细说明
3. 点击「**确定**」创建

#### 注意事项

- 数据集名称应简洁明了，便于识别
- 创建后可在版本管理中添加多个版本

---

### 4.2 版本管理

**用途**：管理数据集的多个版本，支持数据划分和增强配置。

#### 创建版本

1. 在右侧数据集详情中点击「**新建版本**」按钮
2. 填写版本信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 版本名称 | 版本的标识名称 | v1.0.0、baseline |
| 训练集比例 | 训练集占总数据的比例 | 0.9（90%） |
| 验证集比例 | 验证集占总数据的比例 | 0.1（10%） |
| 测试集比例 | 测试集占总数据的比例 | 0.0（0%） |
| 随机种子 | 数据划分的随机因子 | 42 |
| 包含类别 | 该版本使用的标注类别 | 可多选 |

3. 点击「**确定**」创建版本

#### 版本状态说明

| 状态 | 含义 |
|------|------|
| 准备中 | 版本正在初始化 |
| 就绪 | 版本可以用于训练 |
| 生成中 | 正在生成 YOLO 格式数据 |

---

### 4.3 添加图片到版本

**用途**：向特定版本中添加训练图片和标注信息。

#### 操作步骤

1. 在版本列表中点击版本的「**管理图片**」按钮
2. 在弹出的窗口中：
   - **选择来源**：可按图片上传来源筛选（手动上传/ADB截图/视频抽帧）
   - **选择图片**：点击图片进行选择，可多选
   - **点击全选**：快速选择所有可见图片
   - **数据划分**：选择这些图片归属于哪个集合（训练集/验证集/测试集）
   - **YOLO 类别**：选择该版本使用的标注类别
3. 点击「**确定**」添加图片

#### 筛选说明

| 来源 | 说明 |
|------|------|
| 手动上传 | 通过图片管理页面上传的图片 |
| ADB 截图 | 通过 ADB 连接的设备截图 |
| 视频抽帧 | 从视频中提取的关键帧 |

---

### 4.4 数据增强配置

**用途**：配置数据增强策略，扩充训练数据量。

#### 增强策略

| 策略 | 说明 | 效果 |
|------|------|------|
| 水平翻转 (flip) | 将图片左右翻转 | 2x 数据量 |
| 旋转 (rotate) | 按指定角度旋转 | 增加变化 |
| 缩放 (scale) | 按比例放大或缩小 | 适应不同尺寸 |
| 亮度调整 (brightness) | 调整图片明暗 | 适应不同光照 |
| 对比度调整 (contrast) | 调整图片对比度 | 增强特征 |
| 添加噪声 (noise) | 添加随机噪点 | 提高鲁棒性 |
| 高斯模糊 (blur) | 应用高斯模糊 | 模拟远景 |

#### 参数说明

| 参数 | 范围 | 说明 |
|------|------|------|
| 旋转角度 | 0° - 180° | 图片随机旋转的角度范围 |
| 缩放范围 | 0.5x - 2.0x | 图片缩放的比例区间 |
| 亮度调整 | -50% - +50% | 图片亮度调整范围 |
| 对比度调整 | -50% - +50% | 图片对比度调整范围 |
| 噪声强度 | 0 - 50 | 随机噪点的添加强度 |
| 模糊半径 | 0 - 10 | 高斯模糊的程度 |

---

### 4.5 版本对比

**用途**：对比两个版本的配置和数据差异。

#### 操作步骤

1. 点击版本列表上方的「**版本对比**」按钮
2. 选择要对比的两个版本（版本 A 和 版本 B）
3. 系统显示对比表格，包括：
   - 图片数量
   - 已标注数量
   - 版本号
   - 各指标差异（绿色表示更好）

---

### 4.6 生成 YOLO 格式

**用途**：将版本数据导出为标准 YOLO 格式，可用于模型训练。

#### 操作步骤

1. 在版本列表中点击版本的「**生成 YOLO**」按钮
2. 系统开始生成并下载压缩包，包含：
   - `images/`：所有图片文件
   - `labels/`：所有标注文件（YOLO 格式 .txt）
   - `data.yaml`：数据集配置文件

#### YOLO 格式说明

YOLO 格式的标注文件每行代表一个目标：
```
<class_id> <x_center> <y_center> <width> <height>
```
所有坐标值都是相对于图片尺寸的归一化值（0-1）。

## 五、完整使用示例

### 场景：准备游戏界面检测模型的数据集

**目标**：创建数据集，添加游戏截图和标注，生成训练用的 YOLO 数据

#### Step 1：创建数据集

1. 点击「**新建数据集**」
2. 填写：
   - 名称：`game_ui_v1`
   - 描述：游戏界面元素检测数据集
3. 点击确定

#### Step 2：创建第一个版本

1. 点击「**新建版本**」
2. 填写：
   - 版本名称：`baseline`
   - 训练集比例：0.9
   - 验证集比例：0.1
   - 测试集比例：0.0
   - 包含类别：选择所需的类别

#### Step 3：添加训练图片

1. 点击版本的「**管理图片**」
2. 选择要使用的图片（可通过来源筛选）
3. 数据划分选择「训练集」
4. 点击确定

#### Step 4：配置数据增强

1. 点击「**数据增强**」按钮
2. 选择增强策略：
   - 水平翻转：开启
   - 旋转：15°
   - 亮度：±10%
3. 点击确定保存配置

#### Step 5：导出训练数据

1. 点击版本的「**生成 YOLO**」按钮
2. 等待生成完成
3. 下载生成的压缩包
4. 压缩包可直接用于训练管理模块

## 六、常见问题

### Q1：如何选择数据划分比例？

**建议**：
- 小数据集（< 1000张）：训练集 80%，验证集 20%
- 中等数据集（1000-10000张）：训练集 90%，验证集 10%
- 大数据集（> 10000张）：训练集 95%，验证集 5%

### Q2：数据增强会不会影响标注？

**说明**：数据增强是对图片的几何变换，标注框会同步变换，保持正确对应关系。

### Q3：版本之间有什么区别？

**说明**：
- 不同版本可以包含不同的图片集合
- 不同版本可以有不同的数据划分
- 不同版本可以有不同的增强配置
- 便于进行实验对比

### Q4：已标注和未标注的图片有什么区别？

**说明**：
- **已标注**：图片有关联的目标框标注，可用于训练
- **未标注**：图片没有标注，仅作为数据扩充

### Q5：如何删除数据集？

**操作**：
1. 在数据集列表中找到要删除的数据集
2. 点击右侧的「**删除**」按钮
3. 在确认弹窗中点击「**删除**」
4. 数据集及其所有版本将被删除

## 八、数据模型说明

数据集管理涉及以下核心数据：

| 字段 | 说明 |
|------|------|
| id | 数据集唯一标识 |
| name | 数据集名称 |
| description | 数据集描述 |
| created_by | 创建者 |
| created_at | 创建时间 |

版本相关数据：

| 字段 | 说明 |
|------|------|
| id | 版本唯一标识 |
| dataset_id | 所属数据集 ID |
| version_name | 版本名称 |
| version_number | 版本号 |
| status | 版本状态 |
| train_ratio | 训练集比例 |
| val_ratio | 验证集比例 |
| test_ratio | 测试集比例 |
| image_count | 图片总数 |
| annotated_count | 已标注图片数 |
