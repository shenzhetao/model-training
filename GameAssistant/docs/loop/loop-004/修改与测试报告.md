## 二、本次实现的功能

### 2.1 ADB Wrapper (`app/wrappers/`)

- **ADBWrapper**：封装 adb 命令，提供截图、设备列表、点击、滑动等接口
  - 3 次重试 + 指数退避
  - `capture_to_bytes()` 返回 PNG 原始字节（不写临时文件）
  - `get_connected_device()` 自动查找第一个已连接设备
- **DetectorWrapper**：包装 `ElementDetector`，统一推理接口
- **TemplateWrapper**：独立包装 `TemplateMatcher`

### 2.2 ADB API (`/api/v1/adb/`)

| 端点 | 方法 | 说明 |
|---|---|---|
| `/adb/status` | GET | 检测 ADB 连接状态 |
| `/adb/devices` | GET | 列出所有已连接设备 |
| `/adb/screenshot` | GET | 截取屏幕，返回 PNG 流 |
| `/adb/inference` | GET | 截图 + 推理，返回检测结果 |

### 2.3 Inference Playground（推理测试页面）

- 设备状态检测与展示（ID/型号/分辨率）
- 截图预览（支持刷新）
- 三种推理模式切换（混合/仅模板/仅 YOLO）
- YOLO 置信度阈值滑块
- 检测结果表格展示（类别/坐标/置信度/来源）
- 自动刷新（2~30 秒可配置间隔）
- 检测结果 JSON 导出
