## 四、风险提示

### 🔴 高风险

**1. wrappers/ 依赖 GameAssistant 原代码**

- API 服务需要能访问 `GameAssistant/pc/core/` 和 `GameAssistant/config/`
- Docker Compose 需要正确 volume 挂载

**2. ADB 环境依赖**

- 平台运行机器需要安装 `adb` 命令行工具
- 需要检测 `adb` 是否可用并给出友好提示

### 🟡 中风险

**3. 推理性能**

- YOLO 推理在 CPU 上较慢，可能需要 GPU 支持
- 推理 API 暂未做并发限制
