### 1.4 技术要点说明
> **chromium-fresh 配置优化**：Playwright 配置文件中添加 `testIgnore` 选项，明确指定哪些测试文件不应在 chromium-fresh 项目中运行，解决了需要 auth 状态的测试在无 auth 状态下运行的问题。

> **Modal 选择器改进**：将 Modal 弹窗标题选择器从 `.ant-modal-title` 改为 `.ant-modal .ant-modal-title`，避免 strict mode violation 错误。

> **弹窗关闭方式统一**：统一使用 `keyboard.press('Escape')` 关闭 Ant Design Modal，避免因按钮文本匹配失败导致的测试失败。
