## 二、本次实现的功能

### 2.1 后端 API

| 端点 | 方法 | 说明 |
|---|---|---|
| `/annotations/classes` | GET | 列出所有标注类别 |
| `/annotations/classes` | POST | 创建类别 |
| `/annotations/classes/{id}` | PUT | 更新类别 |
| `/annotations/classes/{id}` | DELETE | 删除类别 |
| `/annotations/images/{id}` | GET | 获取图片所有标注 |
| `/annotations/` | POST | 创建单个标注 |
| `/annotations/batch` | POST | 批量创建/更新标注 |
| `/annotations/{id}` | PUT | 更新标注 |
| `/annotations/{id}` | DELETE | 删除标注 |
| `/annotations/by-image/{id}` | DELETE | 删除图片所有标注 |
| `/annotations/projects` | GET/POST | 项目列表/创建 |
| `/annotations/projects/{id}` | GET/PUT/DELETE | 项目详情/更新/删除 |
| `/annotations/projects/{id}/images` | POST | 添加图片到项目 |
| `/annotations/projects/{id}/images` | DELETE | 从项目移除图片 |
| `/annotations/export/yolo` | POST | 导出 YOLO 格式 ZIP |
| `/annotations/stats` | GET | 标注统计 |

### 2.2 Annotation Manager 页面

- **项目标签页**：项目列表、状态筛选、进度展示、CRUD 操作
- **标注标签页**：图片选择、画布标注（Bounding Box 绘制）、类别面板、标注列表
- **类别标签页**：YOLO 类别管理（名称/颜色/YOLO ID/快捷键）
- **统计概览**：总标注数/手动/自动/已完成项目

### 2.3 标注画布功能

- 画框模式（draw）：鼠标拖动画矩形，创建标注
- 编辑模式（edit）：点击标注区域删除
- 浏览模式（view）：只读显示所有标注框
- 标注自动保存到后端
