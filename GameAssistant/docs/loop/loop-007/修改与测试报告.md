## 二、本次实现的功能

### 2.1 Model Manager

- 模型列表（架构筛选、统计概览）
- 模型上传（.pt/.onnx/.tflite）
- 模型详情（指标、类别、标签管理）
- 设为默认/活跃模型
- 模型删除

### 2.2 Training Manager

- 训练任务列表（状态筛选、进度展示）
- 发起训练（选择数据集版本 + 架构 + 全套 YOLO 参数）
- 训练详情（实时日志流 SSE、Loss/mAP 曲线）
- 停止训练
- 训练历史和日志

### 2.3 后端 API

**模型 API：**

| 端点 | 方法 | 说明 |
|---|---|---|
| `/models/` | GET/POST | 列表/上传 |
| `/models/{id}` | GET/PUT/DELETE | 详情/更新/删除 |
| `/models/{id}/activate` | POST | 设为默认 |
| `/models/stats/overview` | GET | 统计 |

**训练 API：**

| 端点 | 方法 | 说明 |
|---|---|---|
| `/training/` | GET/POST | 列表/创建 |
| `/training/{id}` | GET/PUT | 详情/更新 |
| `/training/{id}/stop` | POST | 停止训练 |
| `/training/{id}/logs` | GET | SSE 日志流 |
| `/training/{id}/metrics` | GET | 所有 epoch 指标 |
| `/training/{id}/progress` | GET | 当前进度 |
