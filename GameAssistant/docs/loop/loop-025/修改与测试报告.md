### 1.4 技术要点说明

> 本次修改涉及的核心技术决策和实现细节。

**技术方案**：
1. 使用 `unplugin-vue-components` 插件配合 `AntDesignVueResolver` 实现 Ant Design Vue 组件的自动按需导入
2. 使用 `unplugin-auto-import` 插件实现 Vue Composition API 的自动导入（ref, computed, onMounted 等）
3. 保留 `ant-design-vue/dist/reset.css` 样式导入（必须，用于组件样式）

**配置说明**：
- `AntDesignVueResolver({ importStyle: false })`：禁用内联样式导入，由 CSS reset 文件提供基础样式
- `AutoImport`：自动导入 vue、vue-router、pinia 的常用 API，减少手动导入
