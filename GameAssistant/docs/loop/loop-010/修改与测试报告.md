- **ECharts 评估曲线**：选择模型后加载其关联训练任务的日志，渲染 Box Loss / Cls Loss / mAP50 / mAP50-95 四条曲线，双 Y 轴（Loss 和 mAP）

- **Precision/Recall 曲线**：训练期间 P/R 变化趋势
- **日志下载**：训练详情页 Tab 栏提供下载按钮，导出 .log 文件
