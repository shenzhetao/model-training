## 二、本次实现的功能

### 2.1 Template Manager

- **模板目录树**：按 class_name 分组展示
- **模板上传**：支持 PNG/JPG/BMP/WEBP，自动保存到 `templates/{class}/` 目录
- **模板预览**：缩略图 + 详情展示
- **模板启用/禁用**：热切换匹配状态
- **模板匹配测试**：选择图片 + 模板 + 阈值，返回匹配位置和置信度
- **按类别筛选 + 搜索**

### 2.2 Dataset Manager

- **数据集 CRUD**：创建/删除数据集
- **版本管理**：每个数据集支持多个版本（v1.0, v1.1...）
- **添加图片**：从已有图片中选择，指定 train/val/test 划分
- **YOLO 生成**：生成标准 YOLO 格式 ZIP（含 dataset.yaml + train/val/test 目录）
- **进度统计**：显示各划分的图片数量和已标注数量

### 2.3 后端 API 端点

**模板 API：**

| 端点 | 方法 | 说明 |
|---|---|---|
| `/templates/classes` | GET/POST | 类别 CRUD |
| `/templates/` | GET/POST | 模板列表/上传 |
| `/templates/{id}` | GET/PUT/DELETE | 模板详情/更新/删除 |
| `/templates/{id}/serve` | GET | 模板图片流 |
| `/templates/test` | POST | 模板匹配测试 |

**数据集 API：**

| 端点 | 方法 | 说明 |
|---|---|---|
| `/datasets/` | GET/POST | 数据集列表/创建 |
| `/datasets/{id}` | GET/PUT/DELETE | 数据集详情/更新/删除 |
| `/datasets/{id}/versions` | GET/POST | 版本列表/创建 |
| `/datasets/{id}/versions/{vid}` | GET/PUT | 版本详情/更新 |
| `/datasets/{id}/versions/{vid}/images` | POST | 添加图片 |
| `/datasets/{id}/versions/{vid}/stats` | GET | 版本统计 |
| `/datasets/{id}/versions/{vid}/generate-yolo` | POST | 生成 YOLO ZIP |
