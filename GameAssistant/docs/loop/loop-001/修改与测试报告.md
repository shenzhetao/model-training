### 1.3 技术要点说明
> 本次修改涉及的核心技术决策和实现细节。

1. **FastAPI 生命周期管理**：使用 `lifespan` 上下文管理器管理数据库连接和应用启动/关闭事件
2. **JWT 认证**：采用 OAuth2PasswordBearer 方案，支持 form-data（`/login`）和 JSON（`/login/json`）两种登录方式
3. **数据库迁移**：使用 Flyway 进行 SQL 脚本迁移，V1__initial_schema.sql 包含 21 张表及初始数据
4. **Docker 编排**：MySQL + Backend + Frontend 三服务编排，共享网络，支持健康检查
5. **路由守卫**：前端使用 `router.beforeEach` 进行登录状态检查，未登录自动跳转登录页
6. **默认管理员账户**：admin / admin123（密码已哈希存储在初始迁移中）

## 四、备注

### 4.1 快速启动指南

**方式一：Docker 启动（推荐用于生产）**
```bash
cd web
cp .env.example .env  # 编辑 .env 配置数据库密码
docker-compose up -d
```

访问：
- 前端：http://localhost:3000
- 后端 API：http://localhost:8000
- API 文档：http://localhost:8000/docs

**方式二：本地开发**
```bash
# 后端
cd web/backend
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 前端
cd web/frontend
npm install
npm run dev
```
