### 1.4 技术要点说明

#### Bundle 分包策略 (manualChunks)
```typescript
manualChunks: {
  'vue-vendor': ['vue', 'vue-router', 'pinia'],
  'ant-design': ['ant-design-vue'],      // 1.4MB -> 439KB gzipped
  'echarts': ['echarts', 'vue-echarts'], // 520KB
  'vueuse': ['@vueuse/core'],            // 单独分包（当前为空，因 @vueuse/core 本身已 tree-shake）
  'axios': ['axios'],                    // 45KB
}
```

#### 图片懒加载实现
- 使用 Intersection Observer API
- 预加载阈值 rootMargin: '200px'（提前 200px 开始加载）
- 图片进入可视区域后才加载真实 URL
- 使用 shimmer 动画占位符提升用户体验

#### API 缓存策略
- 请求级缓存：GET 请求自动缓存（5分钟 TTL）
- 存储级缓存：图片列表缓存（30秒 TTL）
- 失效机制：上传/删除操作后自动清除相关缓存
- 支持手动失效：`invalidateCache()` / `clearRequestCache()`
