### 1.4 技术要点说明

1. **视频上传 API** (`POST /api/v1/videos/upload`)
   - 支持 MP4/AVI/MKV/MOV/WMV/FLV/WebM 格式
   - 使用 OpenCV (cv2) 读取视频元信息（时长、尺寸、帧率）
   - 视频存储在 `uploads/videos/` 目录

2. **视频抽帧 API** (`POST /api/v1/videos/{id}/extract-frames`)
   - 三种抽帧策略：interval（按时间间隔）/ count（固定数量）/ scene_change（场景变化检测）
   - 使用 BackgroundTasks 后台执行，不阻塞请求
   - 抽帧结果写入 `images` 表，source 标记为 'video'
   - 支持 MD5 去重，跳过重复帧

3. **前端视频抽帧 UI**
   - 使用 Ant Design Collapse 折叠面板
   - 视频上传 + 列表展示
   - 抽帧参数配置（策略选择 + 参数输入）
   - 实时任务状态轮询（每 2 秒）
   - 抽帧完成后自动刷新图片列表

4. **数据库表**
   - `source_videos`：存储视频元信息
   - `video_extraction_tasks`：跟踪抽帧任务状态
   - 注意：表已在 `V1__initial_schema.sql` 中定义
