## 三、风险提示

### 🔴 高风险

**1. wrappers/ 目录依赖链断裂**

- Inference Playground（推理测试）依赖 wrappers/
- wrappers/ 依赖 GameAssistant 原代码（`../GameAssistant/ml/`, `../GameAssistant/pc/`）
- 如果目录结构变化，需要同步更新 import 路径

**2. ADB 环境依赖**

- 平台运行机器需要安装 `adb` 命令行工具
- 需要检测 `adb` 是否可用并给出友好提示
- 不同操作系统 adb 路径不同（Windows/macOS/Linux）

**3. 视频大文件上传内存问题**

- 当前实现一次性读取整个视频到内存
- 10GB 视频会导致内存爆炸
- 需要尽快优化为流式处理

### 🟡 中风险

**4. 前后端联调风险**

- 视频 API 端点 URL 需确认与前端一致
- 前端 `videos.ts` 中 `getServeUrl` 返回 `/videos/{id}/serve`（无 /api/v1 前缀）
- 需确认 `request.ts` 基础 URL 配置

**5. 数据库迁移风险**

- 所有表已在 `V1__initial_schema.sql` 中定义
- 如果执行了部分迁移，后续 Flyway 脚本可能需要调整
