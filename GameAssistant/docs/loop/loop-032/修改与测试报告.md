### 1.4 技术要点说明

> 本次修改涉及的核心技术决策和实现细节。

1. **全局重试机制**：
   - 将 `retries` 从 0 改为 1（本地环境）和 2（CI 环境）
   - 提高测试稳定性，减少偶发失败的影响

2. **超时配置**：
   - 测试级别超时：30 秒
   - expect 断言超时：10 秒
   - 适用于网络较慢或后端响应延迟的场景

3. **用户流程测试**：
   - 依赖 Playwright storageState 自动处理登录状态
   - 测试页面导航、模态框、路由守卫等核心功能

4. **标注管理测试**：
   - 测试标注项目的增删改查
   - 测试 Tab 切换功能
   - 测试类别管理和审核管理
