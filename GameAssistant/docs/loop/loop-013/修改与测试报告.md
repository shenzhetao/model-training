## 二、本次实现的功能

### 2.1 移动端适配

- **抽屉导航**：`<768px` 隐藏顶部菜单，改为左侧 Drawer，Logo + 菜单项，点击菜单项自动关闭
- **响应式断点**：`hide-mobile` / `hide-desktop` 工具类，resize 监听自动关闭抽屉

### 2.2 图片懒加载

- **Intersection Observer**：只加载视口内 + 200px 预加载区域内的图片
- **Shimmer 占位符**：图片未加载时显示骨架屏动画
- **缩略图**：使用 `?size=400` 参数加载小图，减少带宽
- **响应式网格**：`minmax(180px, 1fr)` → `minmax(140px, 1fr)` → `2列`，全平台适配

### 2.3 表格响应式

- **DatasetManager**：左右分栏改为响应式（`<md` 单列堆叠）、Descriptions 响应式列数、`scroll="{ x: 600 }"` 横向滚动
- **图片选择**：Modal 改为 Drawer，更适合移动端
