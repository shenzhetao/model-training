## 二、本次实现的功能

### 2.1 全局 UI 优化

- **设计 Token**: 完整的 CSS 变量体系（颜色/字体/间距/阴影/圆角/动画）
- **骨架屏**: `skeleton-shimmer` 动画 + 多种 skeleton 尺寸类
- **空状态**: 统一空状态组件样式（含图标/标题/描述/操作按钮槽位）
- **加载状态**: 居中加载、内联加载、页面级加载
- **动画**: fade-in/slide-up/scale-in + Vue transition 类 + staggered children
- **工具类**: flex/text/shadow/spacing/responsive 工具类
- **页面过渡**: BasicLayout router-view 淡入淡出切换

### 2.2 BasicLayout 增强

- 顶部刷新按钮（带旋转动画）
- 页面切换淡入淡出过渡（`out-in` 模式）
- 移动端响应式（隐藏 logo 文字和用户名）
- Sticky header + 阴影
- 用户下拉菜单美化

### 2.3 Template Manager 增强

- **大图预览**: 点击预览区域弹出大图 modal
- **阈值色彩**: 进度条颜色随阈值变化（绿/蓝/黄）
- **阈值提示**: 根据阈值给出"严格/平衡/宽松"文案
- **匹配方法**: 支持 TM_CCOEFF_NORMED / TM_SQDIFF 等方法选择
- **多匹配开关**: 允许返回多个匹配结果

### 2.4 Dataset Manager 增强

- **数据增强配置**: 旋转/翻转/缩放/亮度/对比度/噪声/模糊 7 种策略，每种可调参数
- **版本对比视图**: 选择两个版本并排对比图片数/已标注数/版本号，差异值高亮
