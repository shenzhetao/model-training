### 1.4 技术要点说明
> **Tab 切换测试优化核心**：
> 1. 使用 `[role="tab"]` 属性选择器替代 `.ant-tabs-tab` 类选择器，因为 Ant Design Vue 的 Tabs 组件会渲染 `role="tab"` 属性
> 2. 验证方式从检查 CSS 类 `ant-tabs-tab-active` 改为检查 `aria-selected="true"` 属性，更符合 ARIA 规范且更可靠
> 3. 新增 `switchToTab` 辅助函数统一 Tab 切换的等待逻辑，减少代码重复
